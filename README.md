# Get-Bussiness-information-
Get bussiness information from Google Maps for you own Use, Using openwebninja
---

In the following link, I created the full instruction of getting information from google Map using Google API: https://github.com/hamed-tgh/Get-Bussiness-information-using-Google-Places-New-

---

However, if you want to get information of bussinesses from google map you must know that google APIs are not free, so one other choice could be openwebninja. 

In this project we are going to get information we want by using openwebninja, www.openwebninja.com. 

In this repositort you can find bussunesess information using their name, location and specific area.

This framework offers a wide range of Plans. However, in this project we use its free plan by which we can get about 500 Businesses / Month. 




# Requirements
1) csv
2) time
3) http.client
4) json
